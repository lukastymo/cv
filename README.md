# Lukasz - CV

## Step 1 - Prerequisites

```
brew install pdflatex pandoc
```

## Step 2 - Edit markdown

Edit CV markdown as you want: https://github.com/lukastymo/cv/blob/master/src/resume.md

## Step 3 - Generate

Generate PDF:

```
./bin/generate.sh
```

The CV is generated in `final/` folder.
